Configurable Rendering Engine for Photo Realistic Ocean Waves

software developer
|
OPENGL Graphic rendering

Realistic animation and rendering of the ocean waves is an important aspect for simulators, movies and video games, where it has been advanced but often with simplified treatment of waves using FFT. In coastal engineering research, recent computer advancement made the direct simulation of phase-resolved ocean waves using Boussinesq-type or even Navier-Stokes model practical with animations still being produced by basic plotting tools. The objective of this project is to bridge this gap, particularly for coastal engineers to be able to generate photo realistic animations of ocean waves from their researches with minimal requirement on computer graphics programming. The code is based on pyglet, a python based gaming development engine using OpenGL graphics. The code requires a configuration file (yaml) for graphics components and takes care the rest automatically. Other 3-D models can also be incorporated through the configuration file, making it a full-fledged rendering engine. Coming soon on Github...

CurWav-Sed

software developer
|
Numerical modeling

This is Aaron's primary research topic at Tongji University. The objective is to develop a state of art numerical modelling system for current wave coupled hydrodynamic and sediment transport model based on a semi-implicit Eulerian-Lagrangian Finite Volume Method. The mesh module handles the triangular grid in Euler System, on which shallow water equations are solved. The convection term is treated using Lagrangian approach and the hydrostatic pressure is handled semi-implicitly. The resulting linear system with sparse coefficient matrix is solved by JCG method. The current module is coupled with wave module SWAN in parallel using the concept of communication file. The convection and diffusion equation for passive substances such as temperature, salinity, sediment etc. are solved by Finite Volume Method and are consistent with the current module. The model is fully validated and successfully applied to several projects, including a salinity intrusion study in Yangtz River and sediment transport modeling for a reclamation project in Bohai Bay. Finite Volume Model for SWEs and Wave Energy Equation

software developer
|
Numerical modeling

This is Aaron's research topic at North Carolina State University. The objective is to develop a numerical model to study instability flows such as shear flow, instability of rip current. It was used to validate his theoretical investigation of generation mechanism for rip current as a result of hydrodynamic instability (versus traditional bathymetry controlled rips). This model also successfully reproduced the results from a laboratory study on shear flow in a partially vegetated open channel conducted by Nepf's group at MIT, capturing the development of shear flow in terms of the momentum thickness, dominant frequency, as well as the equilibrium momentum profile and vortex structure. GIS Based HecRas Tools

software developer
|
arcgis application

In response to FEMA’s First Order Approximation for mass produce estimates of flood hazard boundaries of complex watersheds, Aaron was funded by Stantec Research Grant and developed an ArcGIS based tool using Python/Arcpy to automate some time-consuming process, including: automatic hydrology data fetch from StreamStat website; automatic delineation of stream center lines from terrain data; automatic generation of HEC-RAS flow file with arbitrary number of discharge nodes based on hydrology database and HEC-GeoRAS database; automatic preparation of ArcGIS hydrology and hydraulic spatial database deliverables from hydrology analysis, HEC-RAS 100yr and 500yr floodplain shapefiles, PostRas database, and RASPlot database.
